# Baby Bond Survival Effect

## What it does
Pokemon with the **Baby Bond marker** can sometimes survive a knockout hit at **1 HP**.

When it triggers, the battle text is:

`[Pokemon] held on out of love!`

## Trigger chance
- **25%** chance on a lethal hit.
- Can only trigger **once per battle** per Pokemon.

## Who can use it
- The marker is currently tied to the **Premier Ribbon**.
- If a Pokemon has the Premier Ribbon, it is eligible for this effect.

## Baby species support
- Baby species are automatically marked with the Premier Ribbon by the game logic.
- This marker persists through evolution, so evolved forms keep eligibility.
